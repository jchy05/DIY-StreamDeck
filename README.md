# DIY-StreamDeck
Custom Stream Deck built using an ATmega32U4 Pro Micro, 5 mechanical switches, and 5 potentiometers.

## Features

- Copy
- Paste
- Discord Mute
- Discord Deafen
- OBS Replay Buffer

## Hardware

- ATmega32U4 Pro Micro
- 5 Mechanical Switches
- 5 Potentiometers
